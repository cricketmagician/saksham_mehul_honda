# SAKSHAM HONDA SHOWROOM 🏍️

A premium, cinematic web platform for Saksham Honda Showroom based in Chhattisgarh. Featuring a modern glassmorphic design, interactive bike catalog, EMI calculator, and a functional test ride booking system.

## ✨ Features

- **Premium UI/UX:** Cinematic scroll reveals, glassmorphism, and Apple-inspired design.
- **Interactive Catalog:** Explore 13+ Honda bikes and scooters with real-time color switching.
- **Advanced Specs:** Detailed technical specifications (Max Power, Max Torque, etc.).
- **EMI Calculator:** Real-time loan estimation with interactive sliders.
- **Booking System:** Functional test ride booking integrated with a PostgreSQL (Neon) backend.
- **Trust Signals:** Highlighting 4.5+ Google Business rating.

## 🚀 Tech Stack

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Neon)
- **Deployment:** Vercel

## 🛠️ Setup Instructions

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/cricketmagician/saksham_mehul_honda.git
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file in the root:
   ```env
   DATABASE_URL=your_postgresql_connection_string
   PORT=3000
   ```

4. **Run Local Server:**
   ```bash
   node server.js
   ```

## 🌍 Vercel Deployment

1. Push your code to GitHub.
2. Link your repository to **[Vercel](https://vercel.com)**.
3. Add the `DATABASE_URL` environment variable in the Vercel dashboard.
4. Deploy!
